Links Docs »

Links need to be distinguishable from surrounding text by more than just color when the contrast difference is less than 3:1. If a set of links within an element won't have surrounding text, you can remove the underlines by adding class no-link-underline to the element (rather than adding text-decoration-none to each individual link).

Utility Classes Docs »

Use HTML classes instead of CSS for common layout needs.

Example spacing classes:

  • m-0 - no margin
  • p-0 - no padding
  • mt-1 - a little top margin
  • mt-5 - a big top margin
  • ps-3 - some start padding (left in LTR text)

Example display classes:

  • d-none - don't display
  • d-lg-block - display as block for large screens
  • d-inline - display inline
  • d-none d-xl-block - visible only on xl screens
